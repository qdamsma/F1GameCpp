# F1 game in C++

Deze applicatie bevat een object georienteerd project waar verschillende projecten uit een tutorial samen komen met de toevoeging van een zelf gemaakte f1game.
De f1game heeft een basis principe uit de tutorial genomen en verbeterd met behulp van zelf gemaakte classes.
Zo heeft de auto een draai functionaliteit, een tijd systeem, eigen sprites en een simpele bot die de auto probeert in te halen.
